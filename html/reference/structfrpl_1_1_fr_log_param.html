类成员变量说明

bool frpl::FrLogParam::bTimeHeaderMS

日志信息中时间头是否精确到毫秒，默认为否（即精确到秒）

int frpl::FrLogParam::clearPeriod

日志清理周期,单位为天，默认为180天

int frpl::FrLogParam::consoleThreshold

level大于等于此阈值的会同时输出到控制台，应比levelThreshold大。默认为5

bool frpl::FrLogParam::enableLock

每次写日志是否加锁，默认为加锁

int frpl::FrLogParam::fileMaxSz

文件大小上限，超出此大小会切分文件，默认为2MB

int frpl::FrLogParam::levelThreshold

level大于等于此阈值的才会输出日志，必填

string frpl::FrLogParam::logDirPath

log文件存放的文件夹路径。默认为".\logs"

string frpl::FrLogParam::logFileName

log文件名（一般与模块名一致），不带文件类型扩展名，必填
