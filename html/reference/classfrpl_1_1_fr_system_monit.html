详细描述

Windows操作系统下系统相关信息获取

提供静态函数获取系统信息或状态
提供类函数GetCpuUsage来获取cpu使用率，此函数获取的是两次调用之间时间段的cpu使用率
[跨平台]： 否
[依赖库]：无

注解
在调用QueryDriveType和GetDiskInfo时注意盘符的格式为“X:\”，一般先调用GetDriveStrings获取合法盘符字符串后再调用这两个函数
作者
lws

static bool WINAPI frpl::FrSystemMonit::EnablePrivileges ( )
static

获取系统权限，可配合“以管理员身份运行” vs中设置“以管理员身份运行”：在vs工程属性中，Linker —> Manifest File—> UAC Execution Level，选择requireAdministrator (/level='requireAdministrator')选项 在获取进程信息时，如果程序之前没调用此函数提升权限，则会有很多进程的具体信息获取不到

返回
操作是否成功
bool frpl::FrSystemMonit::GetCpuUsage ( struct FrCpuInfo &  cpuinfo)

获取当前cpu信息（上次调用此函数到此次调用之间时间段的cpu使用率）,由于是依据系统时间和cpu空闲时间来推算的，因此两次调用之间要间隔一段时间(100ms以上） 注意此函数不是静态函数

参数
[in]cpuinfo保存输出结构的结构体
返回
