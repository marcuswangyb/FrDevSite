控制台窗件类 [多线程安全]：是
[跨平台]： 否
[依赖库]：qt4.8.1
控制台使用步骤： 1、建立一个qt GUI程序 2、导入头文件ConsoleWidget.h 3、把ConsoleWidget.ui导入到界面目录下 4、在界面上要放置控制台的部分添加一个Widget,把窗件提升为ConsoleWidget类 5、可选（如果你需要同时使用几种不同样式的控制台，就要这一步，否则，请跳过） 首先调用SetName()方法设置对象名（默认对象名为listWidget），并复制样式表的内容，修改ConsoleWidget.qss里面对象的名字(#后面的部分)，修改{}里面的样式 6、调用LoadStyle()方法设置控制台的样式 7、如果要改变控制台大小，可以在qt设计师里面拉伸或者调用窗件的setGeometry()方法
