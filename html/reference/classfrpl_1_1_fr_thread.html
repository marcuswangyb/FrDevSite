详细描述

线程基类(虚类)

继承类必须实现纯虚函数ThreadMemberFunc，可选择是否重写函数ThreadInitFunc
线程运行逻辑:

ThreadInitFunc();
while(线程未终止)
{
ThreadMemberFunc();
}

[多线程安全]：是
[跨平台]： 否
[依赖库]：无
