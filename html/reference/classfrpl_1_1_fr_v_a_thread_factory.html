详细描述

视频采集线程工厂

生成视频采集线程,该工厂会自动读取指定配置文件中的视频采集模块参数 用法示例：

采集图像：
FrVideoBufParam vp，赋值
FrVideoBuffer buffer，buffer.create(vp)
FrVAThreadFactory factory;
FrVideoAcqThread *acqth = factory.Create(channel); // 由工厂生成指定路的线程
acqth->Initialize(&buffer); // 绑定acq线程与buffer
acqth->StartThread(); // 启动acq线程
读取显示图像(与FrVAFactory一致)：
IplImage *curImg = cvCreateImage(buffer.GetImageSize(),8,3);
FrErrCode retval = buffer.QueryFrame(curImg,tag);
if(retval == frpl::ERR_NO)
{
// show image code
}
