详细描述

数据库操作结果,弃用

数据库操作基础类
使用过程：
1.设置FrConnOpt连接参数
2.调用构造函数FrDBConnBase(FrConnOpt opt)
3.调用FrErrCode Connect(int trytimes = 1)
4.连接成功之后即可进行各项数据库操作
5.不用数据库时为释放资源，需要调用析构函数~FrDBConnBase()
提供对Qt的数据库操作的包装
[依赖库]：Qt

作者
lws

FrErrCode frpl::FrDBConnBase::Commit ( )

数据库事务处理提交，如果数据库支持transaction 如果使用事务，要确保事务执行过程中不能让数据库重连，因为数据库重连会丢失事务信息 确保数据库不重连，也即置trytimes为1

返回

FrErrCode frpl::FrDBConnBase::Connect ( int  trytimes = 1)

连接数据库 qodbc两种连接方式： 1.设置用户数据源DSN，此时可以设置FrConnOpt的dbname为数据源名称，其它选项正常设置 2.设置连接字符串，此时可以设置FrConnOpt的dbname为连接字符串，字符串的格式为： mysql：opt.dbname = "Driver={MySQL ODBC 5.2 Unicode Driver};Server=192.168.1.89;Database=test;" oracle：opt.dbname = "DRIVER={Oracle in oracle-odbc};DBQ=52.91.10.3/ITMP;DBA=W"
