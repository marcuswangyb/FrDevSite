详细描述

视频采集工厂类

生成视频采集模块，若在主线程中调用此工厂，则视频解码采集会工作在主线程 用法示例：

采集图像：
FrVideoBufParam vp，赋值
FrVideoBuffer buffer，buffer.create(vp)
FrAcqParam ap，赋值
FrVAFactory factory;
FrVideoAcq *acq = factory.create(1); // 由工厂生成指定acq
acq->Initialize(&buffer,ap); // 绑定acq与buffer
acq->StartAcq(); // 启动acq
读取显示图像：
IplImage *curImg = cvCreateImage(buffer.GetImageSize(),8,3);
FrErrCode retval = buffer.QueryFrame(curImg,tag);
if(retval == frpl::ERR_NO)
{
// show image code
}
